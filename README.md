# AI Language Coach

AI Language Coach is a **Next.js-based language learning application** powered by **Gemini AI**. It allows users to practice languages through interactive AI conversations, track their progress, view chat history, and download detailed performance reports. The app is built with **Next.js, MongoDB, Tailwind CSS**, and integrated AI to provide a realistic learning experience.

## Features

- **User Authentication**: Sign up, log in, and manage user profiles securely.
- **Interactive AI Conversations**: Practice languages in real-time with an AI agent.
- **Chat History**: Users can view past conversations to track improvement.
- **Progress Reports**: Generate downloadable reports to monitor learning progress.
- **Confidence Building**: Designed to help users improve speaking and comprehension confidence.
- **Responsive UI**: Modern, mobile-friendly design with Tailwind CSS.
- **Full-Stack Tech Stack**: Next.js frontend, Node.js backend, MongoDB database.

## Tech Stack

- **Frontend**: Next.js, React, Tailwind CSS  
- **Backend**: Next.js API Routes, Node.js  
- **Database**: MongoDB  
- **AI**: Gemini AI Agent for interactive conversation  
- **Authentication**: JWT or session-based auth
